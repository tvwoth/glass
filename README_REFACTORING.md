## 🚀 Как использовать

### 1️⃣ Веб-приложение (как было)

```bash
docker-compose up
# Доступно на http://localhost
```

### 2️⃣ REST API (новое)

```bash
curl -X POST http://localhost:5000/api/calculate \
  -H "Content-Type: application/json" \
  -d '{
    "config": {"cd_len": 65, "hcor": 80, ...},
    "input_data": {"n1": 100, "angle_EF": 5, ...}
  }'
```

### 3️⃣ CLI (новое)

```bash
python -m app.cli config.json
```

### 4️⃣ Как Python модуль

```python
from app.core.calculate import calculate

result = calculate(input_data, config)
```

## 🎓 Основные компоненты

### `app/core/calculate.py` - 450+ строк

**Функции:**
- `calculate(input_data, config) → dict` - основная функция
- `_calculate_points_to_i(...)` - расчёт точек A-I
- `_calculate_missing_n1/n2/n4(...)` - поиск недостающих параметров
- `_calculate_missing_angle_D(...)` - поиск угла
- `calculate_points(...)` - финальный расчёт всех точек

**Особенности:**
- ✅ Нет классов, только функции
- ✅ Нет глобального состояния
- ✅ Все параметры явные
- ✅ Бинарный поиск для недостающих параметров
- ✅ Полная поддержка коррекции hcor

### `app/api/routes.py` - REST API

**Endpoint:** `POST /api/calculate`
- ✅ Принимает JSON с config и input_data
- ✅ Вызывает core.calculate()
- ✅ Возвращает результаты в JSON

### `app/cli.py` - CLI интерфейс

**Использование:**
```bash
python -m app.cli config.json
```

- ✅ Читает JSON конфигурацию
- ✅ Вызывает calculate()
- ✅ Выводит результаты в консоль
- ✅ Генерирует график

### `app/utils/plot.py` - Визуализация

**Функция:** `generate_plot(points, output_path)`
- ✅ Генерирует PNG график
- ✅ Отделена от CORE
