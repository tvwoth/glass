## Единый источник математической логики

Единственный файл, содержащий математические формулы и алгоритмы расчёта:

```text
app/core/calculator.py
```

Класс `ContourCalculator` в этом файле содержит:
* Формулы координат точек A–K
* Алгоритмы бинарного поиска (n1, n2, angle_D)
* Валидацию входных параметров
* Обработку знаков и углов

**Все остальные модули являются обёртками или потребителями этого ядра.**

## Работа со знаками

### Конвенция

* Координаты `j_x` и `c_x` всегда отрицательные (левая сторона контура).
* В web-интерфейсе (`app/config_service.py`, функция `parse_h_params`) к введённым
  значениям применяется операция `-abs()`.
* В API и CLI значения передаются как есть — отрицательные.

### Запреты

* Операции `+` и `-` являются частью математической модели. Их упрощение запрещено.
* Изменение знаков без анализа влияния на все точки запрещено.
* Любые преобразования вида `abs()`, `-abs()` должны быть зафиксированы в документации.

---

## Работа с углами

### Конвенция

* `angle_EF` — угол наклона сегмента EF (0°–10°).
* `angle_D = 180 + angle_EF` — угол D вычисляется из angle_EF.
* `alpha = 270 - angle_D` — вспомогательный угол для тригонометрических вычислений.
* Углы задаются в градусах, в формулах конвертируются в радианы через `math.radians()`.

### Запреты

* Направление отсчёта углов изменять запрещено.
* Нормализацию углов изменять запрещено.
* Порядок вычислений `angle_D → alpha` изменять запрещено.

Расположение: `app/rendering/matplotlib_renderer.py`

Функция `render(points, output_path)`:
* Принимает список из 11 кортежей (x, y)
* Сохраняет PNG-файл графика
* Использует matplotlib с бэкендом `Agg`
* Ось Y инвертирована (как на технических чертежах)

## CLI

### Доступные команды

#### `calculate` — Выполнить расчёт

```bash
python -m app.cli calculate <config.json>
```

* **Аргументы:** путь к JSON-файлу конфигурации
* **Действие:** выполняет расчёт и выводит результат в консоль
* **Побочный эффект:** автоматически сохраняет `<config>.result.json`

Пример входного файла:
```json
{
    "config": {
        "j_x": -48, "c_x": -160, "cd_len": 65, "de_len": 20,
        "fg_len": 20, "gh_len": 70, "hi_len": 43.5, "jk_len": 8.5, "hcor": 80
    },
    "input_data": {
        "n1": 100, "n2": null, "n4": null, "angle_EF": 5
    }
}
```

#### `render` — Построить график

```bash
python -m app.cli render <config.json> [output.png]
```

* **Аргументы:** путь к JSON-файлу, необязательный путь для PNG
* **Действие:** выполняет расчёт и сохраняет график

#### `export-pdf` — Экспорт в PDF

```bash
python -m app.cli export-pdf <config.json> [output.pdf]
```

* **Аргументы:** путь к JSON-файлу, необязательный путь для PDF
* **Действие:** выполняет расчёт и создаёт PDF-отчёт

#### `config-list` — Список конфигураций

```bash
python -m app.cli config-list
```

* **Аргументы:** нет
* **Действие:** выводит список системных и пользовательских конфигураций

#### `config-edit` — Создать/изменить конфигурацию

```bash
python -m app.cli config-edit <name> [key=value ...]
```

* **Аргументы:** имя конфигурации, пары `key=value`
* **Действие:** создаёт или обновляет пользовательскую конфигурацию
* **Пример:** `python -m app.cli config-edit MyConfig j_x=-48 c_x=-160 cd_len=65`

## Структура тестов

```text
tests/
├── test_calculate_wrapper.py   # Тесты обёртки calculate()
├── test_contour_calculator.py  # Тесты legacy-реализации

### Ключевые тесты эквивалентности

`test_core_calculator.py` сравнивает результаты `app/core/calculator.py` и
`app/legacy/contour_calculator.py` для всех комбинаций входных параметров.
Точность сравнения: `1e-10`.

```bash
