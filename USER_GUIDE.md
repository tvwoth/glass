# Руководство пользователя

## Назначение приложения

Contour App — это веб-приложение для расчёта геометрии контуров стеклопакетов.
Приложение вычисляет координаты 11 точек контура (A–K) на основе геометрических
параметров стеклопакета и трёх из четырёх заданных размеров.

**Входные параметры (три из четырёх):**
* **n1** — расстояние от верхнего края до точки B
* **n2** — расстояние от точки E до точки F
* **n3** — расстояние от точки I до точки J (вычисляется как n4 − hcor)
* **angle_EF** — угол наклона сегмента EF (0°–10°)

Четвёртый параметр вычисляется автоматически.

---

## Работа через веб-интерфейс

### Выбор конфигурации

1. Откройте приложение в браузере: `http://localhost:5000`
2. В блоке **«Конфигурация»** выберите пресет из выпадающего списка:
   * **задайте значения** — нулевые параметры (ручной ввод)
   * **Премьер комплект YP01YP02YP03G01** — системный пресет
   * **Синдикат КЗ471КЗ472КЗ473** — системный пресет
   * Сохранённые пользовательские конфигурации
3. Нажмите **«Загрузить»** — параметры H заполнятся автоматически.

### Изменение конфигурации

После загрузки пресета можно вручную изменить любые параметры H в соответствующих
полях ввода. Если значения изменены, конфигурация автоматически считается
«Пользовательской».

### Выполнение расчёта

1. Заполните **три** из четырёх полей ввода:
   * `n1`
   * `n2`
   * `n4` (расстояние от точки I до K, включая коррекцию hcor)
   * `angle_EF` (угол, 0°–10°)
2. Оставьте одно поле **пустым** — оно будет вычислено.
3. Установите флаг **REV** (реверс направления), если необходимо.
4. Нажмите **«Рассчитать»**.

### Просмотр результатов

После расчёта отображаются:
* Значения n1, n2, n3, angle_EF
* Параметры H1–H8, HCOR
* Координаты всех 11 точек (A–K)
* График контура

### Работа с графиком

График отображается автоматически после расчёта. Оси:
* **X** — горизонтальное положение
* **Y** — вертикальное положение (ось инвертирована, как в технических чертежах)

Каждая точка подписана буквой (A–K) с координатами.

## Сохранение результатов

### PDF

1. В блоке результатов нажмите **«Экспорт PDF»**.
2. Файл будет скачан в формате PDF с параметрами расчёта, координатами и графиком.

### JSON

```bash
python -m app.cli calculate config.json
# Результат сохраняется в config.result.json
```

Формат JSON:
```json
{
  "n1": 100,
  "n2": 45.23,
  "n3": 40.15,
  "n4": 120.15,
  "angle_EF": 5,
  "angle_D": 185,
  "points": [[0, 0], [0, -100], ...]
}
```

### CSV

```bash
python -c "
from app.core.calculate import calculate
from app.storage.export_csv import export_csv
result = calculate(input_data, config)
export_csv(result, 'output.csv')
"
```

Формат CSV:
```
Параметр,Значение
n1,100
n2,45.23
...
Точка,X,Y
A,0.0000,0.0000
B,0.0000,-100.0000
...
```

## Работа с конфигурациями

### Создание

1. Войдите в **режим редактирования** (введите пароль администратора).
2. Задайте параметры H в полях ввода.
3. Введите имя конфигурации в поле **«Имя конфигурации»**.
4. Нажмите **«Сохранить»**.

Имя может содержать буквы, цифры, пробелы, `_` и `-`. Максимальная длина — 100 символов.
Нельзя использовать имя системного пресета или зарезервированное имя «Пользовательская конфигурация».

### Изменение

1. Загрузите нужную конфигурацию.
2. Измените параметры H.
3. Сохраните под тем же именем (обновление) или под новым (создание копии).

### Дублирование

1. Загрузите конфигурацию.
2. Введите новое имя.
3. Нажмите **«Сохранить»**.

### Удаление

1. Войдите в режим редактирования.
2. Выберите пользовательскую конфигурацию.
3. Нажмите **«Удалить»**.

> Системные пресеты удалить нельзя.

## Типовые сценарии использования

### Сценарий 1: Расчёт с известным углом

**Дано:** n1=100, n2=50, angle_EF=5°. Найти n4.

1. Загрузите конфигурацию «Премьер комплект».
2. Введите: `n1 = 100`, `n2 = 50`, `angle_EF = 5`.
3. Поле `n4` оставьте пустым.
4. Нажмите «Рассчитать».
5. Результат: n4 будет вычислен (≈120.15).

### Сценарий 2: Расчёт длины n1

**Дано:** n2=50, n4=120, angle_EF=5°. Найти n1.

1. Загрузите конфигурацию.
2. Введите: `n2 = 50`, `n4 = 120`, `angle_EF = 5`.
3. Поле `n1` оставьте пустым.
4. Нажмите «Рассчитать».
5. Результат: n1 будет найден методом бинарного поиска.

### Сценарий 3: Расчёт угла

**Дано:** n1=100, n2=50, n4=69.17. Найти angle_EF.

1. Загрузите конфигурацию.
2. Введите: `n1 = 100`, `n2 = 50`, `n4 = 69.17`.
3. Поле `angle_EF` оставьте пустым.
4. Нажмите «Рассчитать».
5. Результат: angle_EF будет вычислен (≈5°).

## Частые ошибки

### «Необходимо ввести ровно три параметра»

Введено меньше трёх или все четыре параметра. Оставьте ровно одно поле пустым.

### «Угол наклона должен быть в диапазоне 0°–10°»

Значение angle_EF должно быть от 0 до 10 включительно.

### «Длина ... должна быть положительной»

Параметры n1, n2, n4 должны быть строго положительными (> 0).

### «Выберите конфигурацию или введите значения H»

Все параметры H равны нулю. Загрузите пресет или введите ненулевые значения H.

### «Не удалось найти длину n1/n2»

Бинарный поиск не сошёлся за 100 итераций. Проверьте корректность заданных параметров.

### «Режим редактирования конфигураций»

Для сохранения, удаления или переименования конфигураций необходимо войти в режим
администратора. Пароль по умолчанию: `admin`.
